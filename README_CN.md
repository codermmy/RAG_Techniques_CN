## 技术详情介绍

### 1. 基础 RAG (Simple RAG) 🌱
- **概述**：适合初学者的最基础检索增强生成技术。
- **实现**：基础的“向量检索 + LLM 生成”流程。

### 2. 命题分块 (Proposition Chunking) ⛓️‍💥
- **概述**：将文本分解为简洁、完整且有意义的“命题”句子。
- **实现**：使用 LLM 从文档块中提取事实陈述，并进行质量评分。

### 3. HyDE (假想文档嵌入) ❓
- **概述**：生成一个假想的答案文档，并使用它的嵌入向量去检索真实文档。
- **实现**：通过 LLM 生成“看起来像正确答案”的文本，解决查询与数据不匹配的问题。

### 4. 上下文分块标题 (Contextual Chunk Headers) 🏷️
- **概述**：在分块前添加文档或章节级的标题上下文。
- **实现**：提高检索在特定章节内的准确性，防止断章取义。

### 5. 语义分块 (Semantic Chunking) 🧠
- **概述**：根据文本的含义而非固定长度来划分文档块。
- **实现**：利用 NLP 技术识别文档中的主题边界，确保每个分块的完整性。

### 6. 融合检索 (Fusion Retrieval) 🔗
- **概述**：结合关键词搜索（BM25）和向量搜索。
- **实现**：使用互惠排名融合（RRF）算法合并不同检索来源的结果。

### 7. 重排序 (Reranking) 📈
- **概述**：对初步检索到的结果进行更精准的相关性打分。
- **实现**：使用交叉编码器（Cross-Encoders）或 LLM 对候选文档进行二次排序。

### 8. Graph RAG (知识图谱集成) 🕸️
- **概述**：利用知识图谱中的结构化关系来丰富检索内容。
- **实现**：从文本中提取实体和关系，结合非结构化文本提供更全面的答案。

### 9. RAPTOR (递归抽象处理) 🌳
- **概述**：通过递归摘要构建树状结构的索引。
- **实现**：对文档进行分层概括，支持对大范围文档内容的宏观和微观检索。
